# Person Do Thing Game
See the official game at
- https://persondothing.com/
- https://want.persondothing.com/

This is an unofficial recreation with multi language (+learning) support.

Motivation behind this is to have a fun way to learn another language by playing with translations.

Since this uses the basic words over and over again this is perfect for trying to describe something you don't know the word for in another language using basic words!

This is a personal project for fun, if you like the game, support the original creators at the links at the top.
